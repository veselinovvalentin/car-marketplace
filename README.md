# CarMarket

CarMarket is a simple web application for buying and selling cars between private users.  
Owners can post listings, manage their own cars, and admins can approve or reject submissions.

---

## Tech Stack

### Backend

- **Language**: Java
- **Framework**: Spring Boot (Spring MVC)
- **Persistence**: Spring Data JPA
- **Validation**: Jakarta Bean Validation (e.g. `@NotNull`, `@NotBlank`, `@Min`)

- **Build Tool**: Maven

### Frontend
